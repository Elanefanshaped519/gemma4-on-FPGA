# RTL Sources

This folder contains the hardware source files included for transparency and replication.

Contents:
- `designs/` - Verilog/SystemVerilog design modules used in this project line
- `formal/` - SystemVerilog formal verification files
- `constraints/` - XDC constraints

Notes:
- This release includes Verilog/SystemVerilog (`.v`, `.sv`) sources.
- No VHDL (`.vhd`, `.vhdl`) sources are currently included in this bundle.
- The deployed artifact line in this repo is the 36-core profile (see root `README.md`).
